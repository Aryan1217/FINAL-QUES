Cypher CODE

1. Optimal Stock Purchase and Sale -
Write a function maxProfit(prices: List[int]) -> int that takes in an array prices and returns the maximum profit you can make.
Example 1:
Input: prices = [7,1,5,3,6,4]
Output: 5
Example 2:
Input: prices = [7,6,4,3,1]
Output: 0
2. Single Number -
Given a non-empty array of integers nums, every element appears twice except for one. Find that single one.
Example 1:
Input: nums = [2,2,1]
Output: 1
Example 2:
Input: nums = [4,1,2,1,2]
Output: 4
Example 3:
Input: nums = [1]
Output: 1
3. Persistence Bugger -
Write a function persistence(num: int) -> int that returns the multiplicative persistence of a number.
Example 1:
Input: 39
Output: 3
Example 2:
Input: 999
Output: 4
Example 3:
Input: 4
Output: 0
4. Armstrong Number -
Given a positive integer N, return true if and only if it is an Armstrong number.
Example 1:
Input: 153
Output: true
Example 2:
Input: 123
Output: false
5. Encrypt -
Encrypt a message by converting the first letter to its ASCII code and switching the second letter with the last.
Example 1:
Input: "Hello"
Output: "72olle"
Example 2:
Input: "hello world"
Output: "104olle 119drlo"
